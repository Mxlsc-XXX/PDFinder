Como Usar o PDFinder

1. Digite o assunto desejado no campo "Assunto".

2. Selecione o idioma dos resultados no campo "Idioma".

3. Ajuste a quantidade de resultados que deseja ver.

4. Clique em "Pesquisar" para iniciar a busca.

5. Os resultados serão exibidos abaixo, com links diretos para os PDFs.
